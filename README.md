# DMall Bot

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/azzpoulet/dmall-bot.git
   cd dmall-bot
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure your environment variables as needed.

4. Run the bot:
   ```bash
   npm start
   ```

## Usage Guide

- To use the bot, ensure it's running and configured properly.
- Follow any specific commands and configurations outlined in the project's documentation.
